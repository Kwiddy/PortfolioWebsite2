Project information

  • Project date: May, 2020
  • Client: Dream Heritage - Beaurepaire Manor

Software Engineering Group Project

For this assignment, myself and a team of five others were tasked with creating an interactive and responsible online mobile hotspot for visitors of the Beaurepaire Priory ruins in Durham. I was a front-end developer for this project and was therefore responsible for the creation and presentation of the website where other members of the group worked on the back-end or the interactive map. Due to this project being for the Dream Heritage CIC, the Github repository for this project has been set to private, however you can see the final product here:
